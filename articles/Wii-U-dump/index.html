Steps to repack Wii U games into .wux files

  1. Format an SD card as FAT32 with a 32KB (32768) allocation using guiformat
  2. Download the latest release of Tiramisu, the Sigpatches Module, and wudd
  3. Extract and copy the contents of the Tiramisu folder into the root of your SD card
  4. Copy the 01_sigpatches.rpx file to /wiiu/environments/tiramisu/modules/setup on your SD card. Create a folder in /wiiu/apps named wudd, then paste the wudd.rpx file into it. The root directory of your SD card should now look similar to this
  5. Insert the SD card into your Wii U
  6. Go to wiiuexploit.xyz on your Wii U browser
  7. Click on Run Exploit!

  10. Reconnect the SD card to your computer
  11. Create a combine.bat file
  12. In the file, write copy /b ^ game.wux.part1 + ^ game.wux.part2 + ^ game.wux.part3 ^ combinedGame.wux (continue for as many parts as your game is split into)
