How to Use Tor with Proxifier

By Alexander Davis

July 10, 2022 | Estimated reading time: 56 seconds

  1. Download and install Proxifier
  2. Download and install Tor Browser
  3. Start Tor Browser and connect to the Tor network. Tor must remain open and connected to the Tor network at all times. If closed out, any apps that are using Proxifier will stop working.
  4. Start Proxifier and click "Continue evaluation". Click on "Profile" in the top menu and select "Name Resolution". Deselect Detect DNS settings automatically, select Resolve hostnames through proxy, and click OK to exit.
  5. Click Profile again and choose "Proxy Servers". Click Add and enter 127.0.0.1 into the Address field and 9150 into the Port field. Select SOCKS version 5 from the Protocol dropdown. Click Check, and you should see a message saying the proxy is ready. Click OK to save. If you get a message asking to use this proxy by default, click no.
  6. Click Profile once more and select Proxification Rules. Double click on the Localhost box, and delete everything in the Target hosts section. Next click Browse under the Applications section and select all apps you want to be proxied. Change Action to Proxy SOCKS5 127.0.0.1 and click OK.

    7. Congratulations, your Tor proxy should now be working through Proxifier!
